# Privacy Policy for ADHD Focus Reader

Effective date: 2026-04-02

ADHD Focus Reader is a Chrome extension that applies ADHD-friendly focus styling to webpage content. This policy explains what the extension accesses, what it stores, and how that information is used.

## Summary

- The extension processes webpage content locally in your browser so it can detect readable sections and apply the reading treatment.
- The extension stores settings and preferences in Chrome extension storage.
- Optional analytics are off by default.
- If optional analytics are enabled, they remain local to your browser and are not sent to a remote server by the current codebase.
- The extension does not sell user data and does not share user data with third parties.

## Information the extension accesses

To provide its core feature, ADHD Focus Reader may access:

- Webpage text and page structure on sites where the extension runs
- The active tab URL and hostname
- Extension settings chosen by the user

This access is used only to analyze the current page, identify readable content, apply the reading treatment, and remember user preferences such as per-site behavior.

## Information stored by the extension

The extension stores the following information in Chrome extension storage:

- Reading settings such as mode, intensity, line spacing, and letter spacing
- Persistence preferences such as per-tab and per-site rules
- Temporary tab-session state needed to keep the reading treatment active in the current tab
- Whether optional analytics are enabled

If optional analytics are enabled, the extension also stores:

- Aggregate counts such as applies, resets, manual-selection starts, manual-selection cancels, and unsupported-page events
- Aggregate counts for reading modes and persistence choices
- Counts of distinct sites and distinct pages derived from local salted hashes

## What the extension does not store for analytics

If optional analytics are enabled, the extension does not store:

- Raw page text
- Selected text
- Raw URLs for analytics
- Personal account credentials

## Data sharing and remote transmission

Based on the current codebase, ADHD Focus Reader does not transmit analytics or webpage content to a remote server.

The extension does not sell, rent, or share user data with data brokers, advertisers, or other third parties.

## How information is used

Information handled by the extension is used only to:

- Detect readable content on the current page
- Apply the reading treatment chosen by the user
- Remember user settings and site preferences
- Show local aggregate usage counts if the user enables optional analytics

The extension does not use handled data for advertising, profiling, creditworthiness, lending decisions, or unrelated third-party purposes.

## User controls

Users can:

- Turn optional analytics on or off from the extension popup
- Clear site-specific rules from the extension popup
- Reset the reading treatment on the current tab
- Remove the extension to delete its local extension data from the browser, subject to Chrome sync behavior for synced settings

## Chrome sync note

Some settings are stored using Chrome sync storage so they can follow the user's signed-in Chrome profile. Sync behavior is controlled by Google Chrome and the user's browser/account settings.

## Security

The extension stores only the data needed to provide its functionality and keeps processing in the browser whenever possible.

## Changes to this policy

If the extension's data practices change, this policy should be updated before those changes are released.
